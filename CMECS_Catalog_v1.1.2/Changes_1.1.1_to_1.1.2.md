# CMECS Catalog Changes v.1.1.1 to 1.1.2

## Summary of Changes
Please note that v1.1.2 addresses _**editorial changes ONLY**_ (unit code edits and typo fixes) and _**DOES NOT**_ include any changes to CMECS units (i.e., concept changes).
- Changes were made to select Biotic Component units and Modifiers to correct Unit Codes in the cmecs.owl and output table files
- Typo corrections were made to misspelled words in the cmecs.owl and output table files  

These corrections have been made in the _**v1.1.2 CMECS Thesaurus**_ (text document of the CMECS Catalog ecological units and definitions, arranged hierarchically, with implementation guidance) but it is otherwise unchanged from the last release. The most recent version of the _**CMECS Thesaurus**_ is:
- [CMECS_Thesaurus_v1.1.2.pdf](https://github.com/NOAA-OCM/cmecs/blob/main/CMECS_Catalog_v1.1.2/CMECS_Catalog_v1.1.2_Thesaurus.pdf)
- [CMECS_Thesaurus_v1.1.2.md](https://github.com/NOAA-OCM/cmecs/blob/main/CMECS_Catalog_v1.1.2/CMECS_Thesaurus_v1.1.2.md)

## CMECS Biotic Component

In v1.1.1, new Unit Codes were assigned to several units where they had been omitted or incorrect (see NOAA-OCM#18). Those codes were erroneously assigned as they had already been assigned to other units. These have now been corrected to:

  - _Biotic Group: Attached Tube-Building Fauna_- unit code change from 2.2.1.24 to 2.2.1.25. Changed in the cmecs.owl file, and csv and xlsx tables. 
    - _Biotic Community: Attached Phoronids_- unit code change from 2.2.1.24.1 to 2.2.1.25.1. Changed in the cmecs.owl file and the csv and xlsx tables
    - _Biotic Community: Attached Pogonophorans_- Unit Code changed from 2.2.1.24.2 to 2.2.1.25.2. Changed in cmecs.owl file and the csv and xlsx tables
    - _Biotic Community: Attached Sabellaria_- Unit Code change from 2.2.1.24.3 to 2.2.1.25.3. Changed in cmecs.owl file and the csv and xlsx tables
    - _Biotic Community: Attached Serpula_- Unit Code change from 2.2.1.24.4 to 2.2.1.25.4. Changed in cmecs.owl file and the csv and xlsx tables
    - _Biotic Community: Attached Serpulorbis_- Unit Code change from 2.2.1.24.5 to 2.2.1.25.5. Changed in cmecs.owl file and the csv and xlsx tables

Unit Code typos were also corrected for the following units (NOAA-OCM#32):
- Stylaster Reef [CMECS_00001279](https://w3id.org/CMECS/CMECS_00001279) changed from 2.1.1.2.1 to 2.1.1.2.2
- Tidal Mangrove Forest [CMECS_00000834](https://w3id.org/CMECS/CMECS_00000834) from 2.8.1.4.8 to 2.8.1.4

- Modifers to Modifiers
- Rhodoliith to Rhodolith
- Herbaceaous to Herbaceous
